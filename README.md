# GMM-based-Color-Image-Segmentation

## Environment
* python 3.9.1
* opencv 4.5.1
* scikit-learn 0.24.1

## Install
```
pip install -r requirements.txt
```

## Train the Models
* run "train.py" to save GMM models as files (M1 & M2)
```
python train.py
```

## Run
* use python interactive window in vscode to run "main.py"
```
python main.py
```

## GMM model selection
* BIC (Bayesian Information Criterion)
